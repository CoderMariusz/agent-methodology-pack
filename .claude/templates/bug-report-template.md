## Severity Guide

| Severity | Criteria |
|----------|----------|
| **Critical** | App crashes, data loss, security vulnerability, core feature broken |
| **High** | Major feature not working, significant user impact, no workaround |
| **Medium** | Feature partially working, workaround exists, moderate impact |
| **Low** | Cosmetic issues, minor inconvenience, edge case only |

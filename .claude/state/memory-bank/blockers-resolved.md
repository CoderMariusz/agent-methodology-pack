## Tips for Good Entries

1. **Include exact error messages** - Makes searching easier
2. **Document the wrong paths tried** - Saves time for future attempts
3. **Be specific in solution steps** - "Update config" is not helpful
4. **Add prevention when possible** - Best fix is avoiding the issue

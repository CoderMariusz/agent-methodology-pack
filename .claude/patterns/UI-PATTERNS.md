## Quick Reference

### Touch Target Sizes

| Element | Minimum | Recommended |
|---------|---------|-------------|
| Button | 44x44px | 48x48px |
| Link | 44x44px | - |
| Checkbox | 24x24px | 44x44px touch area |

### Color Contrast

| Type | Minimum Ratio |
|------|---------------|
| Normal text | 4.5:1 |
| Large text (18pt+) | 3:1 |
| UI components | 3:1 |

### Spacing Scale

```
4px  - micro (between icons)
8px  - small (within components)
16px - medium (between components)
24px - large (sections)
32px - xlarge (major sections)
```
